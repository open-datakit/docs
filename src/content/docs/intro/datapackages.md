## Datapackage structure

A datapackage is a repository made up of individual `.json` configuration files.

A basic datapackage might look as follows:

```
.
├── public/
├── src/
│   ├── assets/
│   ├── content/
│   │   ├── docs/
│   │   └── config.ts
│   └── env.d.ts
├── astro.config.mjs
├── package.json
└── tsconfig.json
```
